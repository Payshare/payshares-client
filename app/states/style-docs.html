Alerts

The alerts directive creates an alert. It generates almost the same markup as bootstrap alerts.

Options:

  • type (string, required): success, info, warning, danger
  • dismissible (string, optional | defaults to false): true, false
  • dismiss-click (expression, optional): Overrides the default functionality of removing the DOM when the dismiss button is clicked. The default removes the whole directive from DOM.

HTML source (AngularJS directive)

<alert type="success">Success</alert>
<alert type="info">Info</alert>
<alert type="warning">Warning</alert>
<alert type="danger">Danger</alert>
<alert type="info" dismissible="true">Dismissible</alert>
<alert type="success" dismiss-click="customFunctionality()">Dismissible with ngClick callback on the dismiss button</alert>

Output

Success Info Warning Danger Dismissible Dismissible with ngClick callback on the dismiss button
